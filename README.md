## Описание
Training Planner — GUI‑приложение для планирования тренировок с фильтрацией, сохранением данных в JSON и поддержкой Git.
### Возможности:
* Добавление тренировок с указанием даты, типа и длительности
* Фильтрация по типу тренировки и дате
* Сохранение данных в формате JSON
* Валидация ввода данных
* Поддержка Git для версионного контроля
## Требования
* Python 3.6 или выше
* Стандартная библиотека Tkinter (встроена в Python)
## Запуск приложения
1. Убедитесь, что установлен Python 3.6+.
2. Скачайте все файлы проекта.
3. Откройте терминал и перейдите в директорию проекта.
4. Запустите приложение:   ```bash   python training_planner.py
